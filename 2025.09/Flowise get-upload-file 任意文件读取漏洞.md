# Flowise get-upload-file 任意文件读取漏洞

# CVE

## 0x01   漏洞简述

Flowise 是一款基于拖拽界面创建自定义大型语言模型（LLM）流程的工具，在其小于 3.0.5 的旧版本中 get-upload-file 存在一个任意文件读取漏洞，未经授权的远程攻击者可以利用该接口读取系统重要文件（如数据库配置文件、系统配置文件等），导致敏感信息泄漏。
