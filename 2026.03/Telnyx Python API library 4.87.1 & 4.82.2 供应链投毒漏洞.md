# Telnyx Python API library 4.87.1 & 4.82.2 供应链投毒漏洞

## 0x01   漏洞简述

Telnyx Python 库为任何 Python 应用程序提供了对 Telnyx REST API 的便捷访问。2026 年 3 月 26 日，互联网上披露其发生供应链投毒攻击，其投毒版本中 `telnyx/_client.py` 被插入了恶意代码，攻击者可借此控制受害者服务器权限。

受影响版本：

```
Telnyx 4.87.1
Telnyx 4.87.2
```
